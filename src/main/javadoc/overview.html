AutoBuilder is a magic builder library for writing unit tests on the JVM.

Main components of the library are:

  • AutoBuilder.java, the entry point class,
  • BuilderDSL.java, the interface describing the main builder API (for clients using the library),
  • ValueResolver.java, the SPI representing the basic building block of value resolution (for extensions customizing the library).

For further information see the description of the individual packages.
